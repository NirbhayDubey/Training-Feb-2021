Function Hoisting

Function hoisting is javascript's default behavior of moving declarations to the top.

//Example 1:

//Declaring Function
function myFunction(){
console.log("Hello world!");
}

//calling function
myFunction();

//Example 2:

//calling function
myFunction2();

//Declaring Function
function myFunction2(){
console.log("Hello world!");
}

Both example will give same output.

Closure

Javascript closure enables a function to have "private" variables.

A closure is a function having access to the parent scope, even after the parent function has closed.
